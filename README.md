## 4. 아키텍쳐

📁 구성
- 숫자 형식만 입력 가능
- 리셋 기능
- 카운터 중복 시작 시 최근 시간으로 업데이트
- 완료 or 과거 날짜 입력 시 카운터 완료 메시지 출력
- 알맞은 형식의 날짜 입력 시 남은 시간 출력
- 브라우저 종료 전까지 데이터 유지
        


## 5. 프로젝트를 통해 배운 점

    - Date 객체를 좀 더 유연하게 다룰 수 있게 되었다.
    - Local Storage를 활용할 수 있게 되었다.
    - setInterval에 대한 깊은 이해를 하게 되었다.
